## Usage

```sh
python3 pypt.py [-h | --help]
python3 pypt.py [-i file.md] [--input=file.md]
python3 pypt.py [-c pypt.conf] [--config=pypt.conf]
# Alternatively
./pypt.py [-h | --help]
```
